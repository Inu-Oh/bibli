This code is a demonstration of Mozilla's Django Tutorial: The Local Library website
https://developer.mozilla.org/en-US/docs/Learn_web_development/Extensions/Server-side/Django/Tutorial_local_library_website
This code is for demonstration not use. Refer to the above link bottom of page for license information.

Features added to original:
- crispy forms, bootstrap5
- font awesome
- librarian functionality to add, edit and delete authors, books, book copies, genres and authors not in admin site but in on pages
- also added forms to change status, lend out and return book copies on webiste, not in admin site
- changed layout and added separate nav items and sub navs to complete above tasks
- added genre and language lists for librarians to easily manage these items
